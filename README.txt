# Giorno Sandwich Bot (Local)

## Étapes pour faire tourner le bot en local :

1. Assure-toi d'avoir Python installé (version 3.10+).
2. Ouvre un terminal dans ce dossier.
3. Crée un fichier `.env` en copiant le contenu de `.env.example` et en mettant tes vraies infos.
4. Installe les dépendances :

    pip install -r requirements.txt

5. Lance le bot :

    python main.py

Tu peux interagir avec lui sur Telegram avec /start.
